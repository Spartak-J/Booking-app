# Booking Oselya Mobile App

React Native (Expo) mobile application for booking housing.

## Stack
- React Native + Expo
- TypeScript
- React Navigation
- Zustand
- Axios

## Run
```bash
npm install
npx expo start
```

## API
The app works through BFF/Gateway.
Configure base URL via environment:

- `EXPO_PUBLIC_API_URL`
- `EXPO_PUBLIC_USE_MOCKS=false`
