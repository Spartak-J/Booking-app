# Данные (mocks) в /src/data

Эта папка — единый вход для мок‑данных, чтобы легко заменить их на реальные API без изменений в UI.

Структура:
- `cities/` — справочник городов (`cities.mock.ts`).
- `hotels/` — отели (`hotels.mock.ts`), владельцы (`owners.mock.ts`), отзывы (`reviews.mock.ts`).
- `bookings/` — пользовательские бронирования (`bookings.mock.ts`).
- `messages/` — сообщения/уведомления пользователя (`messages.mock.ts`).
- `payment/` — сохранённые карты (`payment.mock.ts`).
- `users/` — пользователи (если нужны мок‑профили).
- `favorites/` — избранное (список offerId/userId) — см. `favorites.mock.ts`.

Репозитории в `src/data/*/index.ts` инкапсулируют доступ. В мок‑режиме сервисы читают из этих репозиториев; при подключении API меняем только репозитории.

Важно: моки хранятся в памяти процесса (где есть let‑массивы). При перезапуске приложения данные сбрасываются. Persist‑хранения нет — это ожидаемо для моков.
